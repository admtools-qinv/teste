# Design system input

Cole aqui o design system do app final.

## Itens necessários
- cores primárias/secundárias
- background/surface/text
- tipografia
- raios
- espaçamentos
- sombras
- botões
- campos
- cards
- ícones
- animações

## Formato ideal
Pode ser:
- JSON
- tokens em YAML
- link para Figma exportado
- palette + escala tipográfica

```json
{
  "colors": {},
  "typography": {},
  "spacing": {},
  "radius": {},
  "components": {}
}
```
