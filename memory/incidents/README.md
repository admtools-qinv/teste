# Registro de Incidentes

Diretório para documentação de incidentes de segurança.

## Formato do Arquivo

Nome: `YYYY-MM-DD-titulo-do-incidente.md`

### Campos Obrigatórios

```markdown
# Título do Incidente

- **Data:** YYYY-MM-DD HH:MM UTC
- **Severidade:** Baixa | Média | Alta | Crítica
- **Descrição:** O que aconteceu
- **Ações Tomadas:** O que foi feito para conter e resolver
- **Resolução:** Estado final e confirmação de correção
- **Lições Aprendidas:** O que fazer diferente no futuro
```
