## Lições aprendidas
- Gateway do OpenClaw tem CSP restritivo: zero inline styles/scripts/fonts externas
- Servir arquivos do diretório `control-ui/apps` causa conflito com SPA do Control UI
- Solução: servidor separado na porta 9090 com diretório isolado
- Service workers do Control UI podem interceptar requests em localhost — cuidado
- Aragorn não quer mudar SSH config facilmente — adaptar infraestrutura ao que ele tem
- NODE_PATH=$(npm root -g) necessário para módulos globais
- Gateway tem bug: muda porta no config mas não aplica no restart
- Sub-agents falhando com "pairing required" — investigar
